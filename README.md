# – Tourism Website ✈

A beautifully designed web project built using *HTML, CSS, and JavaScript*,  
that highlights the rich culture, heritage, and breathtaking tourist destinations of India.  

This project showcases how modern web design can be used to promote tourism with elegant visuals and responsive layouts.

---

##  Features
-  Showcases popular tourist destinations across India.  
-  Fully responsive design for mobile and desktop.  
-  Clean, attractive, and user-friendly interface.  
-  Smooth scrolling and modern layout styling.

---

## ⚙ Technologies Used
-  HTML  
-  CSS  
-  JavaScript
